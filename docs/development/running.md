# Running the project


**Get source code**
```bash
git clone https://github.com/crownest/doit-mobile.git  (HTTPS)
git clone git@github.com:crownest/doit-mobile.git      (SSH)
```

**Change directory and branch**
```bash
cd doit-mobile
git checkout staging
```

**Install packages**
```bash
npm install
```

**Run project** 
```bash
react-native run-android
```
